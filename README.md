# Wind-Farm-Optimization
The task to find the location of wind turbines in wind farm such that the total output energy is maximum.

# Project Structure
src/ -> Core MATLAB source files

data/ -> Input data (optional)

# How to Run
1. Open MATLAB
2. Navigate to the `src` directory
3. Run:
       main

# Requirements
MATLAB R2018a or later (recommended)

Optimization Toolbox (if required by extensions)
